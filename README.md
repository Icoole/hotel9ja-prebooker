# Hotel9ja PreBooker

Premium hotel dashboard with glassmorphism design for 2026. Mobile-first responsive app with subscription management, bookings, analytics, and settings.

## 🚀 Quick Start

1. Open `index.html` in your browser
2. App runs immediately - no build required
3. Fully responsive - test on mobile/desktop

## 📱 Features

- Glassmorphism + Neumorphism UI
- Bottom navigation (mobile-first)
- Real-time commission calculator
- Retail vs Subscription plans
- Booking management
- Analytics dashboard
- Settings & profile

## 🎨 Tech Stack

- React 18
- Tailwind CSS
- Lucide React Icons
- Glassmorphism Design System

## Run Command

```bash
# From Desktop, open in browser (Windows)
start hotel9ja-prebooker/index.html
```

Enjoy your premium hotel booking dashboard! 🌟
